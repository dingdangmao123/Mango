# 芒果天气

- 芒果天气是一个简单的天气App,提供城市自动定位，天气查询等功能
- 界面追求简洁，没有多余的功能，采用ButterKnife框架提供注入功能,AndroidChangeSkin框架提供换肤功能
- 天气数据由和风天气提供
- 后续会逐渐完善和优化功能
